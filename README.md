# setup + dotfiles

### Homebrew
```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### Apps
```sh
xargs brew install < apps.txt
```

### Dotfiles
```sh
./dotfiles/install.sh
```
